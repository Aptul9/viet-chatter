# viet-chatter documentation

WhatsApp bot that autonomously replies to a filtered subset of 1:1 chats, with human-like timing, per-person memory, adaptive tone, and dynamic language.

## Two documentation branches

- `user/`: non-technical documentation. Explains what the bot does, how to use it, what to expect. For non-developer readers.
- `dev/`: technical documentation. Architecture, DB schema, scheduler, prompts, edge cases. For anyone touching the code.

## How to read it

- To understand what it does: start from `user/README.md`.
- To work on the project: start from `dev/README.md`.

## Status

v1 shipped (2026-05-16). Single Node project with YAML config + web UI.
